Projects

Here is some of the work I have done

Master's Research

My Master's research project involved executing extensive IoT Penetration Testing. The devices chosen for this project was a small range of Foscam IP Cameras. The intention was to find and report as many vulnerabilities in the cameras' software as possible and explore the similarities and differences in vulnerabilties across devices.
Link to Report

Geospatial Intelligence Inference Pipeline

For this project I developed a high performance computing software pipeline that automated the analysis and classification of satellite imagery at scale to locate and identify areas of interest. I leveraged C++, TensorFlow, OpenCV, Boost, and PostGIS to develop this project to be used for inference processing on remote sensing imagery. It has been used to classify 100,000,000+ data points during production dataset curation.

Senior Capstone

I was the team leader and software architect of my senior capstone project in which we designed and implemented a mobile-oriented web application. This project aimed to improve the efficiency of the ordering and tab system used by restaurants and bars by augmenting this system with technology.
Link to TapX GitHub

MealTracker

Developed and maintain an application written in Java to track the daily meal consumption of over 150 members of Delta Sigma Phi Fraternity. Members swipe their student ID, their meal is recorded in a log which is emailed to specified addresses weekly. Request access to private GitHub repository through the form below.

Cloud-IoT Security Middleware

This project involved the design (and proof-of-concept implementation) of a security middleware that could add intermittent, lightweight, and flexible security at the edge of the IoT network. The resulting paper, Flexible IoT Security Middleware for End-to-End Cloud-Fog Communication, has been submitted to a journal, Future Generation Computer Systems, for publication.
Link to Youtube Presentation

Metamorphic Engine

This project was an independant, undergraduate research study under the guidance of Dr. William Harrison at the University of Missouri. In which I designed and developed a basic metamorphic engine using Python and the PeachPy library in order to create a self-modifying program. Request access to private GitHub repository through the form below.

PSN (Professional Social Network)

This project was part of the Software Engineering group project at the University of Missouri. It is intended, as the name suggests, to be a professional social network aiming to bring professionals and businesses together. It allows users to register and edit their profiles, message each other, view and search for other profiles, and more.
Link to PSN GitHub
